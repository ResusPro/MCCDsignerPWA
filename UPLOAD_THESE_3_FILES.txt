MCCDSigner batch regression harness v0.6.1c
================================================

This hotfix uses NEW filenames. It does not overwrite the old batch-test files
and does not require any service-worker change.

UPLOAD
------
Upload these three files directly to the GitHub repository root:

- batch-test-v0.6.1c.html
- batch-test-v0.6.1c.css
- batch-test-v0.6.1c.js

Then open:

https://mccdsigner.uhded.net/batch-test-v0.6.1c.html?build=061c

The page must show a yellow banner reading:

BATCH HARNESS v0.6.1c — UNCACHED FILESET

If that banner is visible, the new harness is definitely loaded.

The processor inside the page still uses the production MCCDSigner v0.6.1
detection code; only the batch progress interface is v0.6.1c.
